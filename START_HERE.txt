3. 🎨 CUSTOMIZE (Optional)
   → Edit colors in css/styles.css
   → Update GitHub links in index.html
   → Modify text content as needed

4. 🚀 DEPLOY YOUR LANDING PAGE

   OPTION A: GitHub Pages (Recommended)
   ──────────────────────────────────────────
   • Create new GitHub repository
   • Upload landing-page folder contents
   • Settings → Pages → Enable
   • Live at: https://yourname.github.io/repo-name/

   OPTION B: Vercel
   ──────────────────────────────────────────
   • npm i -g vercel
   • cd landing-page
   • vercel
   • Follow prompts

   OPTION C: Netlify
   ──────────────────────────────────────────
   • Drag & drop landing-page folder
   • Instant deployment!
