───────────────────────────────────────────────────────────────────
                    🎯 HOW TO CAPTURE SCREENSHOTS
───────────────────────────────────────────────────────────────────

FROM FLUTTER APPS:

1. Android Emulator:
   ├─ Run your app
   ├─ Click camera icon in toolbar
   └─ Screenshots saved to ~/Desktop

2. iOS Simulator:
   ├─ Run your app
   ├─ Press Cmd + S
   └─ Screenshots saved to ~/Desktop

3. Physical Device:
   Android: Volume Down + Power
   iOS: Side Button + Volume Up

RECOMMENDED WORKFLOW:
1. Open each app (Customer, Delivery, Restaurant)
2. Navigate to key screens
3. Take screenshots of each screen
4. Move to images/ folder
5. Rename according to the guide above
6. Replace placeholder code in index.html
