🎨 Frontend Applications

  • Customer App - Browse & Order Food
  • Delivery Partner App - Accept & Deliver Orders
  • Restaurant App - Manage Menu & Orders
  • Landing Page - Marketing & Information
  • Built with Flutter for iOS & Android

⚙️ Backend Services

  • Main Backend (Port 3000) - Orders, Auth, Payments
  • Delivery Backend (Port 6014) - Live Tracking
  • Restaurant Backend (Port 3001) - Menu Management
  • Node.js + Express.js Framework
  • Real-time Communication via Socket.IO

💾 Database Layer

  • MongoDB Atlas - Cloud Database
  • Stores Users, Orders, Restaurants
  • Real-time Location Data
  • Mongoose ODM for Schema Management
  • Scalable & High Performance

🔌 External Services

  • Firebase - Auth & Push Notifications
  • Razorpay - Payment Gateway (UPI, Cards)
  • Google Maps - Location & Navigation
  • Cloudinary - Media Storage & CDN
  • Email Service - OTP & Notifications
